# Cursor AI 프로젝트 규칙

## 언어 설정
- 모든 커밋 메시지는 **한국어**로 작성합니다.
- 코드 주석도 가능한 한 한국어로 작성합니다.
- 문서 및 README는 한국어를 우선으로 작성합니다.

## 코드 스타일
- TypeScript/React 프로젝트
- shadcn/ui + Tailwind CSS 사용
- Zustand를 통한 상태 관리
- React Router v6 사용

## 프로젝트 구조
- `src/components/` - 재사용 가능한 컴포넌트
- `src/pages/` - 페이지 컴포넌트
- `src/services/` - API 및 비즈니스 로직
- `src/store/` - Zustand 스토어
- `src/utils/` - 유틸리티 함수
